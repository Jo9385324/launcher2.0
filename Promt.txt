## 1. Введение и Видение

### 1.1 О Проекте

**QuantumLauncher** — это следующее поколение лаунчера для Minecraft, который объединяет лучшие функции всех существующих лаунчеров и добавляет уникальные возможности с использованием искусственного интеллекта.

### 1.2 Девиз

> "Всё лучшее в одном месте, сделано лучше"

### 1.3 Ключевые Особенности

| Особенность | Описание |
|-------------|----------|
| **AI Интеллект** | Анализ конфликтов, предсказание FPS, чат-ассистент |
| **Ультимативный Функционал** | Всё от всех лаунчеров в одном |
| **Кастомные Сборки** | Система Forks как в TLauncher |
| **3D Preview** | Анимированная модель персонажа |
| **Безопасность** | Сканер вирусов, верификация |
| **Облако** | Синхронизация везде |

### 1.4 Целевая Аудитория

- **Новички** — один клик, всё работает  
- **Обычные игроки** — удобно и красиво  
- **Продвинутые** — полный контроль  
- **Стримеры** — быстрая смена сборок

## 2. Анализ Конкурентов

### 2.1 Сравнительная Таблица

| Лаунчер | Плюсы | Минусы | Что берём |
|---------|-------|--------|------------|
| **Prism Launcher** | Открытый код, мультиинстансность | Устаревший дизайн, нет AI | Переносимость, стабильность |
| **MultiMC** | Скорость, портативность | Минимализм | Быстродействие |
| **CurseForge** | Огромная библиотека | Тяжёлый, реклама | Интеграция модов |
| **ATLauncher** | Готовые модпаки | Ограниченная кастомизация | Один клик |
| **GDLauncher** | Красивый UI | Electron, закрытый | Дизайн |
| **Lunar Client** | Оптимизация, CPS | Только ванила | Производительность |
| **TLauncher** | Бесплатные моды, Tmods | Небезопасно | Сборки Forks |
| **HMCL** | Много функций | Сложный | Функционал |

### 2.2 Что Объединяем

```
QuantumLauncher = Prism + MultiMC + CurseForge + ATLauncher + GDLauncher + Lunar + TLauncher + HMCL + AI
```

### 3.2 Компоненты Backend

| Модуль | Описание |
|--------|----------|
| `core` | Ядро системы, Quantum Core |
| `api` | HTTP API эндпоинты |
| `ai` | AI система (конфликты, оптимизация) |
| `content` | Моды, шейдеры, ресурспаки |
| `skins` | Управление скинами |
| `forks` | Кастомные сборки |
| `security` | Сканер безопасности |
| `cloud` | Облачная синхронизация |
| `monitoring` | Мониторинг FPS |
| `snapshots` | Резервные копии |
| `integrations` | Java, Auth |

### 3.3 Компоненты Frontend

| Компонент | Описание |
|-----------|----------|
| `FXML Views` | FXML-представления (Scene Builder) |
| `Controllers` | JavaFX Controllers (MVVM-логика) |
| `Services` | Сервисы, репозитории, HTTP-клиент |
| `Themes` | CSS-стили и темы |
| `Components` | Переиспользуемые JavaFX-компоненты |

## 11. Техническая Реализация

### 11.1 Стек Технологий

#### Backend (Java)

| Компонент | Технология |
|-----------|------------|
| Web Framework / API | **Spring Boot 3** |
| Async / Concurrency | Virtual Threads (Java 21+), Project Reactor |
| Database | **Hibernate JPA** + SQLite JDBC |
| Graph Algorithms (зависимости) | **JGraphT** |
| ML/AI | **Deep Java Library (DJL)** / ONNX Runtime Java / LangChain4j |
| Cloud (S3) | **AWS SDK for Java v2** |
| Logging | SLF4J + Logback |
| Dependency Injection | Spring IoC |

#### Frontend

**JavaFX 22+**  
- FXML + Scene Builder  
- CSS (Glassmorphism, анимации)  
- JavaFX Properties & Bindings  
- WebView / FXGL (для 3D Preview при необходимости)  
- Модульная структура (JPMS)  

**Сборка и упаковка:** Gradle (рекомендуется) или Maven + `jpackage` / GraalVM Native Image (опционально)

**Минимальная версия Java:** JDK 21 LTS

### 11.3 База Данных (SQLite)

```sql
-- Инстансы
CREATE TABLE instances (
    id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    version TEXT NOT NULL,
    modloader TEXT,
    path TEXT NOT NULL,
    created_at INTEGER,
    last_played INTEGER
);

-- Моды
CREATE TABLE mods (
    id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    version TEXT NOT NULL,
    source TEXT,
    file_path TEXT,
    instance_id TEXT
);

-- Сборки (Forks)
CREATE TABLE forks (
    id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    author_id TEXT,
    version TEXT,
    content JSON,
    created_at INTEGER
);

-- Скины
CREATE TABLE skins (
    id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    type TEXT,
    path TEXT,
    url TEXT
);

-- Настройки
CREATE TABLE settings (
    key TEXT PRIMARY KEY,
    value TEXT
);
```
